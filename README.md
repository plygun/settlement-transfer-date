## How to run:
```
Install nvm: https://github.com/nvm-sh/nvm/blob/master/README.md
nvm i 10
yarn install
cp .env.example .env
yarn start
```

To run tests you should tap:
```
yarn test
```

Note: 

By default server will listen on 3000 port.

Server port or environment can be modified in .env config file.
